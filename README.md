**Overview :-**

This project demonstrates an end-to-end data analytics workflow starting from raw data processing to business insights and dashboard creation. It includes data cleaning, exploratory data analysis (EDA), feature engineering using Python, SQL-based analysis using PostgreSQL/MySQL, and interactive dashboard building using Power BI. Finally, a professional presentation is created using Gamma for storytelling and insights.

**Dataset :-**

The dataset used in this project is structured and contains key business/employee-related attributes such as ID, Department, Region, Salary, Status, Work Mode, and other relevant fields.

**Tools & Technologies :-**
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (PostgreSQL / MySQL)
Power BI
Gamma AI (Presentation)
Jupyter Notebook

**Project Workflow :-**


1. Data Loading (Python)

The dataset was imported using Pandas and initial exploration was performed to understand structure, data types, and missing values.

2. Data Cleaning (Python)

Missing values were handled, duplicate records were removed, column names were standardized, and data inconsistencies were corrected.

3. Exploratory Data Analysis (EDA)

Performed univariate and bivariate analysis to identify trends, distributions, and relationships between key variables.

4. Feature Engineering

New meaningful features were created to improve analysis. Categorical variables were encoded and derived metrics were added for deeper insights.

5. SQL Analysis

The cleaned dataset was loaded into PostgreSQL/MySQL and SQL queries were performed for:

Aggregations
Group-wise analysis
Filtering and segmentation
Business insights extraction
6. Power BI Dashboard

An interactive dashboard was built including:

KPI Cards (Total Employees, Average Salary, Remote Employees)
Employee Status Distribution
Department Distribution
Region Distribution
Salary and Workforce analysis

Slicers were added for interactivity (Department, Region, Status, Work Mode).
